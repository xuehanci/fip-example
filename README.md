# Fiber Photometry Analysis Tutorial Capsule

This repository serves as the starting point for creating a GitHub-integrated Capsule for preprocessing and exploratory analysis of Fiber Photometry data. 

### Runtime environment

Choose any **Python** starter environment and add the following packages to the mamba package manager:

ipykernel

matplotlib

pandas

seaborn

### Required Data

Running this notebook requires a data asset which can found with the following custom metadata:
- data level = raw
- experiment type = FIP
- subject ID = 655834
